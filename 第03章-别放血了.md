# 第三章 别放血了

回单位的路上，车厢里没人说话。

发动机的声音和来的时候一样闷，但少了一层东西——少了一个人的呼吸。陈铮坐在来时的位置。对面空了。车厢壁上赵德林撞过的地方，那块暗色印子还在，不仔细看已经分不清是血还是蹭上去的脏。

他低头看了一眼右手。虎口下方的热意还在。不是枪响之后那种往上窜的感觉——这一次是稳的。从旧疤正下方铺开，温的，不烫。他把手翻过来，手背正常。又翻回去。虎口表皮看不出任何东西。

他把手放在膝盖上。

到单位的时候已经过了午饭时间。食堂还没收。老杨走在前面，没回头。陈铮跟着进去了。

打菜窗口的灯还开着。铝盆底剩了一层薄汤底，空气里有洗洁精和剩菜混在一起的味道。头顶的灯管有一根在闪，不快，每隔几秒抖一下。

陈铮拿了一个托盘。没什么胃口，还是打了一份饭。托盘里的菜没什么热气——青菜蔫了，豆腐皮边缘发了干。他端着托盘找位置。

窗口阿姨看了他一眼。"猪血汤要不要？今天剩得多。"

陈铮点了一下头。阿姨舀了一碗放进他托盘里。汤面泛一层暗红，铁锈味顺着热气飘上来，很薄，但直接进了鼻子。

陈铮的胃没有任何预兆地猛地收了一下。

不是慢慢来的反胃。是突然一下——从胃底往上撞，像被一只手攥紧了。

他把托盘搁在最近的桌上，转身往洗手间走。步子不快，但每一步都在压。喉咙里已经涌上来了。

洗手间的门推开。他弯下腰，撑着洗手台。

胃里的东西翻上来。不多——本来就没吃。吐出来的大多是酸水。喉咙烧了一下，然后又是一下。眼泪跟着出来了，不是哭，是胃收缩得太猛。鼻腔里全是那股铁锈味，比刚才浓得多。

他拧开水龙头。凉水的声音把别的声音盖掉了。他看着水流把池子冲干净，又接了一捧泼在脸上。

嘴里的味道还是酸的。

他直起腰。镜子里的人脸色有点白。他用袖子擦了一下嘴，站了几秒，深吸了一口气。鼻腔里的铁锈味还没散。

回去了。

桌上那碗猪血汤不见了。

老杨坐在对面，面前没有汤碗。他在吃自己的那份饭。筷子夹起一撮青菜，嚼了。没看陈铮。

陈铮坐下来。老杨没说话。他也没问。

两个人把饭吃完。

下午的事不多。执行当天剩下的主要是整理——表格、签字、交接。陈铮把该填的填完，该交的交了。握笔的时候虎口的热意还在。不是枪响之后那种往外冲的力，是稳稳地铺在皮下。从旧疤下方一直暖到手腕。

他偶尔用左手碰一下右手虎口。皮肤表面没有明显温度，但皮下的热意一直没退。两种感觉叠在一起，说不上来哪里不对。

他继续写。

下午慢慢过去。

三点多，西边的太阳斜进来，纸面晃得发白。陈铮起身去拉窗帘，指尖碰到玻璃。

玻璃是凉的。他的手指也凉，贴上去时几乎分不出差别。

虎口里面还是热的。

下午四点多，他收拾了桌子。表格写完了，笔放回笔筒里。他站起来的时候，右手的温度让他停了一下——不是不舒服，是分神。那点热意待得太久了，久到他已经习惯。习惯之后突然意识到自己习惯了，反而觉得不对。

下班前，他去了趟淋浴间。

淋浴间没有别人。水打在瓷砖上，声音很干净。他打开水，一开始就是凉的。

落在肩膀上。没有平时那一瞬间的收紧。

他顿了一下。把脸凑上去。冷水冲在脸上，皮肤的收紧慢了半拍才到。不是不冷——是身体的反应比平时迟了。他把手伸到水流底下。五根手指很快变凉，指尖的颜色淡了下去。

但他不觉得冷。

水还在冲。他站着。虎口的热意没有因为冷水消退。

一种说不清楚的矛盾——外壳是凉的，里面有一块是热的。

回到宿舍。

窗帘没拉。外面的天已经暗成一片深蓝。灯没开。

陈铮在床边坐下来。他把今天的事一件件排开——肠胃不舒服大概是因为没吃什么东西，冷热感觉可能只是累了，呕吐也许是枪响之后延迟的反应。虎口发热……这个解释不了。

他低头看着右手。

虎口里面是热的。

五根手指却是凉的。
