# 《法警·行刑之后》完整创作总纲与设定集


## 一、核心概念

| 项目 | 内容 |
|---|---|
| **故事时间** | 2009年 |
| **故事地点** | 虚构省份省会城市 |
| **总字数** | 约25—35万字 |
| **总章节** | 约80—100章 |
| **核心案件数** | 5案（第一案15章，后续每案约15—20章） |

**一句话概括**：
2009年，一个刚参加工作的法院法警新人，因执行死刑枪决而觉醒上古刑官血脉，从此白天押解犯人、夜晚处决邪祟。他在行刑牌残魂的指引下，与背负家族宿命的殡葬世家传人相遇，在司法体系与神鬼交界之间，走一条以自身为代价的道路。

**核心矛盾**：每一次处决邪祟都在推进他自身的异化。能力越强，离“陈铮”越远。

**全书题眼**：“他每杀一个‘不该存在的东西’，自己就离‘人’远一步。”

**全书双核主题**：
- 陈铮的线：一个被推入非人境地的人，始终以法警身份守住边界
- 白芷的线：一个背负家族债务的人，最终让那笔债在自己手里结束


## 二、人物系统

### 2.1 主角：陈铮

| 项目 | 内容 |
|---|---|
| **初始身份** | 报考公安系统刑侦岗位未录取，通过省级公务员考试进入省会中院法警支队，入职三个月 |
| **警校背景** | 成绩优异，射击、体能、格斗、战术课程名列前茅。本想去一线刑侦，阴差阳错来了法院 |
| **性格底色** | 闷，但不木。话少是因为脑子里一直在转，不是反应慢。不抱怨，不诉苦，不主动开口。做事有“底下的一层温度”，只是不浮在表面 |
| **说话风格** | 短句，语速不快。紧急时会更短——不是慌，是把所有力气都收拢到必须说出来的那几个字上 |
| **核心动机（执行死刑）** | 客观上支队人手不足；经济上津贴“多一笔算一笔”；异地恋女友关系紧绷需钱维系；老杨有心栽培 |
| **内心深处的恐惧** | 不是死，是“有一天我真的不记得自己是谁了” |
| **人物弧光** | 迷茫不甘的新人法警 → 被动承受异化的恐惧者 → 守住边界的执行者 |

**陈铮的成长节奏**：
- 第一案：觉醒，确认异常存在，第一次处决，建立“人”与“邪祟”的边界
- 第二案：理解印记不是孤例，接受行刑牌，与白芷建立合作雏形
- 第三案：建立双轨调查模式，确认封印存在，与白芷形成信任关系
- 第四案：拼合线索，确认封印被系统侵蚀，接纳自身身份
- 第五案：完成终极处决，守住最后边界，回到法警岗位

| **性格** | 该问的问，不该问的绝不问。知道很多，但选择不让别人为难。关心不落在点上，落在外围——问的都是“能回答的问题” |
| **说话风格** | 短促、干脆、带人情味但不啰嗦。偶尔用半截话——说一半咽一半 |

| **关键场景** | 推津贴申请表；“等你呼吸稳了再扣”；接枪、退弹、验枪、收好；食堂猪血汤；传行刑牌 |

### 2.4 云（旧行刑人残魂）

| 项目 | 内容 |
|---|---|
| **身份** | 晚清至民国时期旧司法体系中的行刑人——从旧衙门衙役/刽子手，到民国法院执刑吏，跨越斩决到枪决的过渡期 |

| **性格** | 一个累了很久的人。不急，不追问，不主动塞信息。会说“你到了那一步自然就知道了”，因为那是他走过路之后的经验 |
| **与陈铮的关系** | 非血缘关系。他是上一个被选中的人，两人是“同一份使命在不同时代的承担者” |
| **与白家的关系** | 旧衙门时期认识白家先祖，参与过白家封印终局邪祟的仪式 |

| **引导特点** | 碎片化回应，不一次性喂饱；知道白家封印的事，知道白芷太爷那一代，不知道白芷本人及之后的全部 |
| **第一案台词** | “你不是第一个。”“人已经死了。”“出来的不是人。”“它找的是你手里那一份。” |
| **终局位置** | 不显形、不指位、不干涉。梦境外不再出现 |

### 2.5 第一个死囚：赵德林

| 项目 | 内容 |
|---|---|
| **身份** | 主动执行封印解除仪式的人，三十七岁，长期独居，具有反社会人格特征但思维清晰 |
| **作案跨度** | 从第一次作案到最终执行历时较长，具体时间线待司法流程资料核实；期间邪祟印记持续侵蚀外围结构 |
| **信息来源** | 从旧殡仪站封死墙体中获得残缺仪式残页，结合梦境中接收的碎片化影像 |
| **知情程度** | 知道自己在做仪式，知道自己必须被正式处决，但不知道封印内有什么，不知道血脉碎片会转移到执行者身上。执行前一刻才意识到自己可能正在开启某种未知的东西 |
| **核心行为** | 按四个外围封印节点位置实施四起杀人；主动留下证据、停止逃亡；将自己推入司法程序，以执行作为仪式第五步 |
| **主观责任** | 没有被完全控制。邪祟只提供了梦境和篡改后的仪式信息，选择杀人、实施犯罪、主动落网都是他自己的决定 |
| **与邪祟的关系** | 封印中的邪祟感知到他在研究外围规律后，通过梦境篡改仪式内容、诱导他完成步骤。赵德林以为自己在利用裂缝获得新生，实际上在替邪祟撕开封印 |
| **对“碰到了”的认知** | 从残页和梦境中知道仪式最后阶段会发生“交接”，但不知道通过血液完成，不知道血脉碎片的存在。押解途中血液接触时，感觉体内长期附着的东西松动离开，因此意识到“交接”可能提前发生 |
| **三句台词的含义** | “碰到了”：确认仪式存在，对交接提前发生的意外，对自己是否仍是受益者的怀疑；“你走的路跟我相反”：执行前一刻才意识到自己正在开启什么；“你跑不掉了”：他相信陈铮也会被裂缝改变，但不知道陈铮会成为与他完全相反的存在 |
| **与陈铮的镜像关系** | 赵德林认为秩序可以被利用来撕开边界；陈铮认为力量必须被秩序划定边界。赵德林主动抛弃人的身份；陈铮越接近非人越要守住最后部分 |

## 三、血脉体系与底层机制

### 3.1 传承链条

> 蓐收（天之刑神/金神/秋神）→ …… → 云（旧行刑人）→（近百年空白）→ 陈铮

### 3.2 封印三层结构

旧封印分为三个层次：

| 层次 | 内容 | 状态（第一案结束时） |
|---|---|---|
| **第一层：外围约束节点** | 四个位置，限制封印内部力量向外投射 | 四个节点全部失效 |
| **第二层：内部残余节点** | 维持封印结构，阻止被封之物完整显形 | 仍然存在，将被后续容器裂缝逐步破坏 |
| **第三层：封印本体** | 由白家气脉和未完成中心送葬仪式共同维持 | 仍保持约束，但出现不可逆主裂口 |

### 3.3 第一案形成的主裂口

赵德林完成四个外围仪式后成为“第五个位置”。被依法执行时，产生的精神裂缝从四个已失效的外围节点之间贯穿过去，在封印结构上形成第一道不可逆主裂口。

封印本体仍未完全打开，终局邪祟仍受到白家气脉和剩余节点约束。但主裂口使它第一次获得稳定向外渗透印记的通道，封印也从此失去自行恢复的可能。

### 3.4 血脉碎片与邪祟印记的关系

| 类别 | 携带物 | 作用 | 是否产生刑官 |
|---|---|---|---|
| **第一个死囚（赵德林）** | 封印中裹带出的蓐收血脉碎片 + 邪祟印记 | 血脉碎片触发陈铮觉醒；执行贯穿封印主结构 | 是（唯一一个） |
| **后续容器型死囚** | 邪祟印记（仅此一项） | 每个印记被执行后，处决裂缝持续扩大主裂口、破坏剩余内部节点 | 否 |
| **非容器死囚** | 无 | 产生普通残渣 | 否 |

血脉碎片来源于当年封印终局邪祟时遗留在封印内部的蓐收血脉残余。赵德林被执行时，碎片混在脱出的印记中，通过血液接触进入陈铮体内。

### 3.5 觉醒方式（双重触发）

| 条件 | 内容 |
|---|---|
| **第一重：血液接触（种因）** | 押解途中赵德林额角破皮，血液通过陈铮手套虎口破口接触旧疤。血迹中携带的血脉碎片和印记通过皮肤接触进入陈铮体内 |
| **第二重：合法处决（点火）** | 枪响瞬间，血脉碎片被激活——合法处决是唯一能让碎片安全转移并嵌入新宿主的条件 |
| **法理认同（内在条件）** | 扣动扳机时陈铮确认自己执行的是法律而非私欲，血脉认可了他的“执行者”身份 |

血液接触具有偶然性：车辆制动、手套虎口存在细小破口、赵德林额角受伤——三者同时发生。

### 3.6 后续容器型死囚的形成

主裂口形成后，终局邪祟只能将印记投射到已经犯下严重罪行、进入死刑程序但尚未执行的人身上。

印记不会操控侦查、影响证据、改变审判结果、干预复核或决定执行时间。它只是等待。当携带印记的死囚被依法执行时，处决产生的裂缝会被主裂口吸引，用于继续破坏封印内部残余节点。

印记必须在执行前附着，不能在执行结束后再附着。

### 3.7 终局邪祟的双层存在

终局邪祟同时存在于两个层面：

- **白家气脉层面**：维持它返回、附着与再生的路径；
- **物质界域层面**：以污染核心显形，并直接造成现实伤害。

白芷的送走仪式负责切断它与白家气脉的联系；陈铮的最终处决负责斩灭显形后的污染核心。只完成送走，污染核心仍可能残留；只完成处决，它仍可能借白家气脉重新返回。因此两人的动作必须在同一时刻完成，缺一不可。

### 3.8 身体异化三阶段

| 阶段 | 时间 | 表现 |
|---|---|---|
| **一** | 执行后1—7天 | 感官过载、后颈发热 |
| **二** | 执行后8—30天 | 骨骼异响、手指粗糙、握力异常、体表温度明显偏低、不出汗 |
| **三** | 第一到三个月 | 闻见死亡气味、看见黑雾、噩梦、对死亡气息产生饥饿感 |

### 3.9 侵蚀与99%停驻机制

侵蚀本身不可逆，身体异化程度不会回退。但侵蚀进度表示血脉可达到的最高激活深度，而不是陈铮日常全天维持对应症状——部分异化永久保留，极端表现只在刑官状态完全展开时出现。

终局中侵蚀的推进停在99%处，原因是獬豸在处决完成时划出了一条边界——它在入队宣誓时见证过陈铮，终局时不施予、不显形，只确认执行者仍处于法理边界之内。它让刑官形态停在完全非人的门槛前。那些失去的不会再回来（低体温、味觉减退、心跳缓慢、虎口旧疤暗金不变），但最后一点人格没有消失。战斗结束后刑官状态消退，永久侵蚀仍保留，但不影响日常履职。

**獬豸的99%停驻机制前文铺垫**：
- 入队宣誓时獬豸在场（客观场景）
- 第二案末，陈铮路过法院门口时獬豸对“已作出的合法选择”产生短暂的确认感
- 第三案末，陈铮再次路过时獬豸没有回应——读者意识到獬豸不是随时响应，而是在符合条件时才会发出微弱确认信号
- 第四案末，獬豸的确认感重新出现，与第三案的缺失形成对照


## 四、能力体系

## 五、核心道具

### 5.1 行刑牌（双层结构）

#### 表层形态：民国旧警徽

一枚民国时期流传下来的旧警徽。正面徽形严重磨损，不具备任何现行警务证件效力。背面经过多次修补，铆接着一层颜色更深的旧铁衬片。由历代法警或参与执行工作的前辈私下保管传递，象征法警职业和执行职责的延续。

#### 内核实质：行刑牌残件

背面的深色旧铁衬片并非普通修补材料，而是年代更早的行刑凭牌残件。残件经过打磨后被包裹或铆接在旧警徽背后，承担梦境通道、残渣感应和后续能力稳定功能。云的残魂寄宿其中。

#### 传承路径

更早行刑者留下旧牌残件 → 民国时期被包裹进警徽 → 若干代法警或参与执行者私下保管 → 老杨师父 → 老杨 → 陈铮。

#### 职业属性

外层警徽象征现代法警职业的连续传承；内层旧牌残件承载更古老的行刑者体系。老杨和其师父主要按"法警职业旧物"理解并传递它。

#### 老杨知情程度

知道旧警徽由前辈法警传下，听说在特定执行者手中可能发热；不知道背片是行刑牌残件，也不知道完整超自然机制。

#### 佩戴方式

陈铮后续使用普通挂绳将旧警徽贴身佩戴。真正产生作用的是警徽背后的旧铁残件，但第一案前期不向人物和读者直接说明。

#### 功能建立

第一案：连通云的梦境、残渣靠近时发热、稳定第一次短刃具现；第二案：刀身不稳定震颤时，胸前行刑牌给出冷而稳的触感；第三案：陈铮主动用手掌按住胸前行刑牌贴向刀柄根部，建立主动使用功能。

#### 终局功能

陈铮以行刑牌维持人格锚点，以双手虎口祭刀，将短刃展开成长刀。行刑牌在此期间衔在口中。

### 7.4 封印主裂口

赵德林被依法执行时，产生的精神裂缝从四个已失效的外围节点之间贯穿过去，在封印结构上形成第一道不可逆主裂口。

第一案结束时的状态：
- 四个外围约束节点已失效
- 封印出现不可逆主裂口
- 部分内部节点仍维持
- 白家气脉仍在约束封印本体
- 中心送葬仪式残留限制尚未解除
- 终局邪祟不能完整进入现实
- 但主裂口使终局邪祟获得向外渗透印记的通道

### 7.6 旧殡仪站两层痕迹

归位残渣被处决后，现场出现两层痕迹：

**外层痕迹**（赵德林仪式+邪祟污染）：
- 潮湿泥土气味、铁锈腥甜味、焦黑木屑、四条向中心收缩的黑线

**内层痕迹**（白家未完成送葬与封印）：
- 干燥草木灰气味、被水反复冲刷后的浅灰环痕、极少量已褪色红色纤维、与残渣阴冷相反的陈旧残留

陈铮在第一案中无法理解内层痕迹，只记录为“中心位置存在第二种不同来源的气味”。第二案白芷将通过新的现场残留识别同类气息。

### 7.7 地面动线

归位残渣留下的四条黑线明确表现为：外围进入中心。

这意味着归位残渣从四个被破坏的外围节点逐步向中心“归位”。陈铮的初步判断是“赵德林的仪式似乎在把某种东西送回中心”。第四案中将确认旧殡仪站是封印结构的中心接收端，真正的封印核心位于更深层。

### 7.8 赵德林笔记缺失页

赵德林的异志笔记中存在一页被完整撕去的内容。陈铮在第一案中只能发现前后页码连续但装订处有撕除痕迹——前一页记录“四方归位”，后一页开始出现“官断开门”和“脱籍”。

赵德林获得残本时该页已经缺失。他根据梦境和自行理解补足了中间逻辑，把“送走”错误理解成“换身”。

缺失页真正表达的是：中心之人不是借死亡返回，而是以自身承担最后一次送走。终局邪祟向赵德林展示的梦境有意颠倒了这一含义。

回收顺序：
- 第一案：发现缺页
- 第三案：确认白芷手抄册中存在相似句式
- 第四案：补全中心仪式的原意
- 第五案：白芷完成当年没有完成的送走

## 十三、写作边界

1. 所有死囚承担完整犯罪责任，不能被邪祟洗白
2. 被害人不能被写成“罪有应得”或仪式自愿者
3. 不质疑刑事案件定罪事实和正常司法程序
4. 终局邪祟不能控制侦查、审判、量刑和执行结果
5. 血液接触必须具有偶然性
6. 第一次短刃具现必须不稳定
7. 第一次战斗不能写成成熟变身战
8. 第一案只建立“第五个位置”的谜面，不揭露白家
9. 邪术笔记必须真假混杂，不能成为说明书
10. 印记必须在执行前附着，不能在执行结束后再附着
11. 封印必须分层（外围节点、内部节点、本体），为后续案件预留空间


## 十四、行文风格约束

### 14.1 基本规则

**句子与段落**：短句优先但不设硬性数字上限。环境描写长度根据场景速度动态调整。动作场景紧凑，舒缓场景适当展开。

**叙述视角**：严格第三人称有限视角，只写陈铮看到、听到、感觉到的东西。禁止“他不知道的是……”式上帝视角跳转。

**描写方式**：感官优先（冷/烫/腥/闷/沉），动作优先（用动作写情绪，不解释），对话与叙述比例根据场景需要自然分配。

**去AI味禁令**：禁止总结式收尾；禁止华丽五官描写；禁止排比三连；禁止高级词汇堆砌；禁止解释说明；禁止套路化开头。

### 14.2 叙事控制（第一案第6—8章）

第六至第八章不得连续使用档案说明或长篇笔记摘录。赵德林背景应通过陈铮的行动逐步显现：

- 第六章：核心是老杨交行刑牌。赵德林资料只作为背景碎片出现——陈铮等待时看见案情摘要上“旧书收购经历”、物证照片中堆满房间的手抄本，但不解释完整含义
- 第七章：核心是云第一次入梦。梦醒后陈铮发现赵德林留下的湿脚印与物证照片中某种仪式步距相似，线索通过视觉对应出现，不由云解释
- 第八章：陈铮实际展开旧地图，在桌面标注四个地点。信息通过操作呈现——尺子连线、不同年代地图比例不一致、某条已改名旧路、四处地点共同朝向旧殡仪站、笔记存在缺页。陈铮只能得出“四起杀人确实围绕某个中心安排”的有限结论

### 14.3 章节体量提醒

第一案15章结构：
- 第1—3章：执行前中后核心事件（碰血、枪响、呕吐）
- 第4—8章：发现异常、确认残渣、接触行刑牌、云入梦、梳理赵德林背景
- 第9—13章：锁定旧殡仪站、战斗、处决
- 第14—15章：收尾和下一案前兆

**风险在第6—8章**：梳理赵德林背景的部分偏说明性，需将异志研究内容分散到动作和场景中——陈铮翻看旧地图时目光在某处停留更久，而不是直接读一段笔记。

**旧殡仪站场景（第10—13章）可集中**：旧殡仪站是第一次短刃确认、第一次处决、第五件木楔发现、动线确认、白家旧封残留和主裂口活动的核心场景。
