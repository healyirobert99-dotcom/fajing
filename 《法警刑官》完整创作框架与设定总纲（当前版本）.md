# 《法警刑官》完整创作框架与设定总纲（当前版本）


## 一、核心概念

**故事时间**：2009年

**一句话概括**：
2009年，一个刚参加工作的法院法警新人，因执行死刑枪决而觉醒上古刑官血脉，从此白天押解犯人、夜晚处决邪祟。他在行刑牌残魂的指引下，与背负家族宿命的殡葬世家传人相遇，在司法体系与神鬼交界之间，走一条以自身为代价的道路。

**核心矛盾**：每一次处决邪祟都在推进他自身的异化。能力越强，离“陈铮”越远。

**全书题眼**：“他每杀一个‘不该存在的东西’，自己就离‘人’远一步。”

**全书双核主题**：
- 陈铮的线：一个被推入非人境地的人，始终以法警身份守住边界
- 白芷的线：一个背负家族债务的人，最终让那笔债在自己手里结束


## 二、人物系统

### 1. 主角：陈铮

| 项目 | 内容 |
|---|---|
| **初始身份** | 报考公安系统刑侦岗位未录取，通过省级公务员考试进入省会中院法警支队，入职三个月 |
| **警校背景** | 成绩优异，射击、体能、格斗、战术课程名列前茅。本想去一线刑侦，阴差阳错来了法院 |
| **性格底色** | 闷，但不木。话少是因为脑子里一直在转，不是反应慢。不抱怨，不诉苦，不主动开口。做事有“底下的一层温度”，只是不浮在表面 |
| **说话风格** | 短句，语速不快。紧急时会更短——不是慌，是把所有力气都收拢到必须说出来的那几个字上 |
| **核心动机（执行死刑）** | 客观上支队人手不足；经济上津贴“多一笔算一笔”；异地恋女友关系紧绷需钱维系；老杨有心栽培 |
| **内心深处的恐惧** | 不是死，是“有一天我真的不记得自己是谁了” |
| **人物弧光** | 迷茫不甘的新人法警 → 被动承受异化的恐惧者 → 守住边界的执行者 |

### 2. 支队长：老杨

| 项目 | 内容 |
|---|---|
| **身份** | 省会中院法警支队支队长，陈铮的直接领导 |
| **性格** | 该问的问，不该问的绝不问。知道很多，但选择不让别人为难。关心不落在点上，落在外围——问的都是“能回答的问题” |
| **说话风格** | 短促、干脆、带人情味但不啰嗦。偶尔用半截话——说一半咽一半 |
| **角色定位** | 陈铮执行死刑的带教人（现场指导、预备射手）；陈铮的守护者（知道一些事但不多问，师父曾提过“这一行祖上不干净”） |
| **关键场景** | 推津贴申请表；“等你呼吸稳了再扣”；接枪、退弹、验枪、收好；食堂猪血汤；传行刑牌 |
| **关键关系** | 师父是省高院法警总队警官，师父与老方是几十年的老同事 |
| **对陈铮的态度** | 他看陈铮像看一个走夜路的年轻人。不跟上去陪他走，但在路口留了一盏灯 |

### 3. 女友：宋晚

| 项目 | 内容 |
|---|---|
| **性格** | 观察力强。她离开是因为她清楚地看到了“他不需要她了” |
| **分手节点** | 发现陈铮执行后的异常状态和手机搜索记录后，发来长消息提出“冷静一下” |
| **核心台词** | “你以前有什么都会跟我讲。现在你什么都不讲了——不是你不愿意，是你已经不知道该怎么讲了。我在这儿也没有意义。” |
| **后续** | 不会再出现。但陈铮会在某些瞬间想起她说的话 |
| **功能定位** | 象征主角“回不去的普通人生” |

### 4. 云（旧行刑人残魂）

| 项目 | 内容 |
|---|---|
| **身份** | 晚清至民国时期旧司法体系中的行刑人——从旧衙门衙役/刽子手，到民国法院执刑吏，跨越斩决到枪决的过渡期 |
| **形象** | 旧式行刑人的粗粝气质，说话带旧衙门底子的尾音，句子利落 |
| **性格** | 一个累了很久的人。不急，不追问，不主动塞信息。会说“你到了那一步自然就知道了”，因为那是他走过路之后的经验 |
| **与陈铮的关系** | 非血缘关系。他是上一个被选中的人，两人是“同一份使命在不同时代的承担者” |
| **与白家的关系** | 旧衙门时期认识白家先祖，参与过白家封印终局邪祟的仪式 |
| **出场方式** | 梦境对话；行刑牌旧铁触感发烫指引；画面闪现 |
| **引导特点** | 碎片化回应，不一次性喂饱；知道白家封印的事，知道白芷太爷那一代，不知道白芷本人及之后的全部 |
| **终局位置** | 不显形、不指位、不干涉。梦境外不再出现 |

### 5. 第一个死囚（仪式发起人）

| 项目 | 内容 |
|---|---|
| **身份** | 一个主动执行封印解除仪式的人。他的作案是按照特定步骤排列的，每一步对应封印结构中的一个节点，最终通过自己被合法处决来贯穿主裂口 |
| **作案跨度** | 从第一次作案到被执行，跨越一年多。封印在这一年多里持续被撕开，到枪响那一刻到达临界点 |
| **信息来源** | 从旧遗址获得的仪式残本，以及梦境中接收的碎片化影像。他误以为完成仪式能获得力量或摆脱死亡，执行时才发现自己被利用 |
| **知情程度** | 他知道自己在做一场仪式，知道自己最终会被处决，但他不知道自己释放的是什么，更不知道血脉碎片会在执行时转移到陈铮身上。他在行刑前说的“你走的路跟我相反”——是在执行前一刻才真正意识到自己正在开启什么 |
| **三句台词** | 押解车上碰血后：“……碰到了。”刑场验身时：“你走的路，跟我相反。我走完了。你才开始。”枪响前一秒：“……你跑不掉了。” |
| **中枪后** | 嘴角的弧度，是任务完成后的释然——他以为一切已经结束，不知道自己打开了什么 |
| **终局回收** | 第五案中，白芷翻查手抄册时会发现一条关于“贯穿封印主节点”的记录，第一案的作案方式与这条记录完全对应 |

### 6. 老方

| 项目 | 内容 |
|---|---|
| **身份** | 省高级人民法院档案室管理员 |
| **性格** | 安静、谨慎、做事极有条理。话不多，但记性好得惊人。知道的比说出来的多，但他只递卷宗，不递判断 |
| **说话风格** | 声音偏低，语速均匀，几乎没有情绪波动 |
| **与老杨的关系** | 老杨的师父（省高院法警总队警官）与老方是老同事、老熟人 |
| **能力** | 能调阅全省各中院的死刑复核卷宗目录、摘要和合规范围内的材料；能发现案件之间的关联模式 |
| **对陈铮的帮助** | 提供案件线索和卷宗摘要；发现异常关联；提供历史旧案参照 |

### 7. 白芷

| 项目 | 内容 |
|---|---|
| **身份** | 民间殡葬世家传人，白事手艺传了多代 |
| **家庭背景** | 父母在她很小的时候去世/失踪，由姑姑带大。没见过父母，对家族身世了解不全面，一直在主动追寻 |
| **性格** | 有耐心，但不磨叽。跟亡者打交道久了，反而比普通人更干脆。跟陈铮说话不绕弯子 |
| **说话风格** | 平和，带一点南方口音的尾调。短句居多，偶尔冒出一句民俗老话 |
| **家族宿命** | 白家当年封印了终局邪祟，但未能完成送走。白芷是面对这个未完成节点的人 |
| **对宿命的认知** | 知道家里“欠过什么”，但不知道全貌，一直在拼凑自己家族的秘密。姑姑告诉她一部分，剩下的靠她自己查 |
| **核心道具** | 姑姑传给她：一本翻烂了的手抄册（部分页被撕掉）、一根旧红绳（连接白家气脉）、一块看不出形状的旧木牌 |
| **与陈铮的关系** | 长期合作伙伴但各自独立行动——他负责处决，她负责送走。后期发展为“知道彼此在同一个方向上走的人” |
| **终局行动** | 完成送走仪式的最后一步，切断邪祟与白家气脉的连接，与陈铮同时行动。红绳断裂，白家宿命结束 |
| **核心台词** | “我们家有一条规矩：做完白事，回来洗澡，水要流干才能走。” “我第一次见到你的时候，我就知道你是那个人。不是我猜的，是我手腕上的红绳在动。” |

### 8. 姑姑

| 项目 | 内容 |
|---|---|
| **身份** | 白芷的姑姑，白家白事手艺的传人，白芷的监护人 |
| **性格** | 她知道很多，但选择性告诉白芷——不是刻意隐瞒，是有些事情“说了也没用，得你自己撞上了才能懂” |
| **她知道什么** | 知道白家祖辈和刑官系统的旧关联；知道白家当年封印过什么东西；知道白家的“宿命” |
| **她没说什么** | 没告诉白芷父母的真相；没告诉白芷那个被封印的东西可能会找回来 |
| **给白芷的东西** | 手抄册、旧红绳、旧木牌 |


## 三、血脉体系与底层机制

### 1. 传承链条

> 蓐收（天之刑神/金神/秋神）→ …… → 云（旧行刑人）→（近百年空白）→ 陈铮

### 2. 血脉碎片与邪祟印记的关系

| 类别 | 携带物 | 作用 | 是否产生刑官 |
|---|---|---|---|
| **第一个死囚** | 封印中裹带出的蓐收血脉碎片 + 邪祟印记 | 血脉碎片触发陈铮觉醒；印记贯穿封印主结构 | 是（唯一一个） |
| **后续容器型死囚** | 邪祟印记（仅此一项） | 每个印记被执行后，其处决裂缝持续扩大主裂口、破坏剩余封印节点 | 否 |
| **非容器死囚** | 无 | 产生普通残渣 | 否 |

### 3. 第一个死囚与后续容器的关系

第一案中，第一个死囚被执行时，封印主结构被第一次贯穿，形成不可逆的主裂口。终局邪祟仍然受到白家气脉和其他封印节点的约束，无法完整进入现世。后续附着有印记的死囚被处决时产生的裂缝，会继续扩大主裂口并破坏剩余节点。五案的关系是：第一案主裂口出现，第二案发现印记并非孤例，第三案发现印记与白家旧事有关，第四案剩余封印节点接连失效，第五案邪祟完成显形，双方终结它。

### 4. 觉醒方式（双重触发）

| 条件 | 内容 |
|---|---|
| **第一重：血液接触（种因）** | 押解车上第一个死囚额头破皮，陈铮手背蹭到血迹。血迹中携带的血脉碎片和印记通过皮肤接触进入陈铮体内 |
| **第二重：合法处决（点火）** | 枪响瞬间，血脉碎片被激活——合法处决是唯一能让碎片安全转移并嵌入新宿主的条件 |
| **法理认同（内在条件）** | 扣动扳机时陈铮确认自己执行的是法律而非私欲，血脉认可了他的“执行者”身份 |

### 5. 封印松动的完整因果链

终局邪祟于晚清至民国时期被白家与云联合封印，封印的力量来源是白家气脉和特定仪式结构。多年后，第一个死囚从旧遗址获得仪式残本，结合梦境接收到的影像信息，开始按照特定步骤作案——每一步对应封印结构中的一个节点。被捕后至执行前的一年多里，封印持续被侵蚀，裂缝逐步扩大。执行当天，合法处决的烈度与封印最后节点对齐，主结构被贯穿，形成不可逆主裂口，邪祟印记从封印缝隙中渗出，裹带出当年封印时遗留在内部的一小块蓐收血脉碎片，通过血液接触进入陈铮体内。

### 6. 身体异化三阶段

| 阶段 | 时间 | 表现 |
|---|---|---|
| **一** | 执行后1-7天 | 感官过载、后颈发热 |
| **二** | 执行后8-30天 | 骨骼异响、手指粗糙、握力异常、体温34℃、不出汗 |
| **三** | 第一到三个月 | 闻见死亡气味、看见黑雾、噩梦、对死亡气息产生饥饿感 |

### 7. 侵蚀与99%停驻机制

侵蚀本身不可逆，身体异化程度不会回退。但侵蚀进度表示的是血脉可达到的最高激活深度，而不是陈铮日常全天维持对应症状——部分异化永久保留，极端表现只在刑官状态完全展开时出现。终局中侵蚀的推进停在99%处，原因是獬豸在处决完成时划出了一条边界——它在入队宣誓时见证过陈铮，终局时不施予、不显形，只确认执行者仍处于法理边界之内。它让刑官形态停在完全非人的门槛前。那些失去的不会再回来（低体温、味觉减退、心跳缓慢、虎口旧疤暗金不变），但最后一点人格没有消失。战斗结束后刑官状态消退，永久侵蚀仍保留，但不影响日常履职。


## 四、能力体系

### 1. 基础能力（被动）

| 能力 | 表现 |
|---|---|
| **死亡嗅觉** | 闻到邪祟特有的铁锈+湿土+腥甜味 |
| **黑雾视觉** | 看到附着在人/物上的黑色煞气 |
| **刑官威压** | 面对邪祟时散发压迫感 |
| **不死体质** | 伤口愈合快于常人 |

### 2. 核心能力（主动）

| 能力 | 表现 | 限制 |
|---|---|---|
| **刑具具现** | 凝聚黑铁短刃/中刀（后期在祭刀时成长） | 初期时灵时不灵 |
| **刑官状态** | 变身后速度、感知能力、观察力大幅提升 | 峰值深度随使用推进，战斗结束后消退 |
| **断罪之眼** | 看“业”的颜色（金/灰/黑/暗红） | 需看眼睛 |
| **最终处决** | 目标彻底消亡不残留 | 每次处决都推进异化 |

### 3. 高阶能力

| 能力 | 内容 |
|---|---|
| **刑场构筑** | 划定区域，目标无法逃逸 |
| **判词宣告** | 对目标说出判决，产生压制 |
| **血脉共鸣** | 借用历代刑官残影的力量或经验 |

### 4. 七重刑台

七重刑台是作者内部使用的阶段控制刻度，不在正文中以数值形式出现。正文中用虎口颜色、体温变化、味觉减退、旧伤状态等具象指标呈现成长阶段和异化进度。

### 5. 武器进化路径

| 阶段 | 形态 | 持刀方式 |
|---|---|---|
| **初期** | 短刃/中刀 | 单手 |
| **中期** | 中长刀 | 单手为主，偶用双手 |
| **终局（祭刀后）** | 完整长刀 | 双手 |

### 6. 变身系统

**常规仪式**：咬破右手虎口旧伤→血渗入掌心浮出的刀柄纹路→后颈发烫→战装覆盖

**终局祭刀（全书高潮）**：

陈铮扯下挂在颈间的行刑牌，衔在口中——旧铁的凉意和气味是他在双手祭刀过程中维持“我是陈铮”这一自我认知的锚点，防止刑官意识完成完全覆盖。然后以双手虎口托住刀刃，双手合十夹住刀刃缓慢向外拉开，血沿着刀刃两侧向下流淌，刀身在血线覆盖区域持续延伸，被拉成长刀形态；血纹走完整个刀身后握刀斩落。行刑牌在此期间始终衔在口中。


## 五、核心道具

### 1. 行刑牌

| 项目 | 内容 |
|---|---|
| **表层形态** | 现代法警证章形态——金属牌，刻有编号、法院名称、警徽图案。和普通法警证章没有明显区别，但与正式警务证件完全分离，作为保存于老杨私人手中的传承物 |
| **内核实质** | 背面嵌入一块旧铁片——从更早的行刑牌残件打磨后嵌入。比周围金属颜色更深，边缘有旧刻痕残留 |
| **年代跨度** | 从晚清或更早的行刑牌，到民国改铸，再到现代法警证章形态 |
| **传承路径** | 旧衙门行刑人→云→省高院法警总队→老杨师父→老杨→陈铮 |
| **职业属性** | 行刑者身份凭证的私人纪念物——历代佩戴者都是参与行刑的人 |
| **佩戴方式** | 陈铮在接受刑官身份后将其挂在胸前，贴住心口，不再取下 |
| **残魂寄宿** | 云的残魂寄宿于背面的旧铁片中，只有陈铮握住它时才有通道连通 |
| **功能建立** | 第二案刀身不稳定震颤时，胸前行刑牌给出冷而稳的触感，建立“旧铁能稳定刀形”的感知；第三案陈铮主动用手掌按住胸前行刑牌贴向刀柄根部，建立主动使用功能 |
| **终局功能** | 陈铮以行刑牌维持人格锚点，以双手虎口祭刀，将短刃展开成长刀。行刑牌在此期间衔在口中 |

### 2. 獬豸

| 项目 | 内容 |
|---|---|
| **存在形式** | 法院门口石雕 |

## 九、终局事件序列

1. 白芷感知到白家气脉震动，独自前往旧封印地点，完成送走仪式准备
2. 金乌从固定位置飞走，引导陈铮到达白芷所在位置
3. 终局邪祟显形。它不是原始旧物，也不是白家传人，而是两者经封印融合后的结果——保留被污染前的记忆，但意志已不再属于原初的那个人
4. 终局邪祟说出最后一块白家历史：当年白家为什么选择“封住”而不是“送走”
5. 白芷完成送走仪式的最后一步，切断邪祟与白家气脉的连接
6. 陈铮双手祭刀：扯下颈间行刑牌衔在口中，双虎口夹住刀刃向外拉开，血沿刀身流淌，短刃在血线中长成长刀
7. 陈铮完成处决，邪祟消亡
8. 红绳断裂，白家宿命结束
9. 侵蚀推进停在99%。不是意志力，也不是运气，是獬豸在处决完成时确认执行者仍处于法理边界之内，它让刑官形态停在完全非人的门槛前。那些失去的不会再回来，但最后一点人格没有消失
10. 战斗结束后刑官状态消退，永久侵蚀保留，但不影响日常履职
11. 终局画面：两个人坐着等天亮。白芷获得选择权，陈铮回到法警支队


## 十、结局后状态

陈铮回到法警支队，继续穿那身制服。老杨开始有意识地将更多组织、协调和带教事务交给他。数年后，可以将老杨把办公室钥匙交给他作为尾声的内容。

他仍然每天路过法院门口那尊獬豸石雕时放慢脚步，伸手碰一下它的角。行刑牌挂在胸前，不再取下。那些永久失去的东西不会回来，但他在法警支队的位置仍然在。他继续做他的工作，继续执行那些应该由他来的死刑。

## 十二、行文风格

**句子与段落**：短句优先但不设硬性数字上限。环境描写长度根据场景速度动态调整。动作场景紧凑，舒缓场景可以适当展开。

**叙述视角**：严格第三人称有限视角，只写陈铮看到、听到、感觉到的东西。禁止“他不知道的是……”式上帝视角跳转。

**描写方式**：感官优先（冷/烫/腥/闷/沉），动作优先（用动作写情绪，不解释），对话与叙述的比例根据具体场景需要自然分配，不强制替代。

**去AI味禁令**：禁止总结式收尾；禁止华丽五官描写；禁止排比三连；禁止高级词汇堆砌；禁止解释说明；禁止套路化开头。
